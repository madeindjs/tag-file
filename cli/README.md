# CLI tool for tag-file

```
Usage: tag-file [options] [command]

Options:
  -V, --version                    output the version number
  -h, --help                       display help for command

Commands:
  toggle [options] <filepaths...>  Add or remove the tags in the given filenames
  add [options] <filepaths...>     Add the tags in the given filenames (if tag already exists, nothing happens).
  remove [options] <filepaths...>  Remove the tags in the given filenames (if tag already exists, nothing happens).
  help [command]                   display help for command
```
